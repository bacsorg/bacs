\page problem_page Problem

\section problem_sec Problem

BACS support different problem formats.
Each problem is a directory tree.

\subsection problem_format_ssec Problem format
In the root directory "format" file is provided.
It's trimmed content is a problem format.

\warning It is not recommended to use problem formats with whitespaces.

\subsection problem_type_ssec Problem type
The first part of problem format, separated by '#' if present, whole format string otherwise,
is problem type. bacs::problem::importer implements particular problem type conversion.
